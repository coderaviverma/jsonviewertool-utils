# JSONViewerTool Utils

Lightweight Java utilities for JSON validation and processing.

## Links
- GitHub: https://github.com/coderaviverma/jsonviewertool-utils
- Maven Central: https://repo1.maven.org/maven2/com/jsonviewertool/jsonviewertool-utils/

## Installation

```xml
<dependency>
  <groupId>com.jsonviewertool</groupId>
  <artifactId>jsonviewertool-utils</artifactId>
  <version>0.1.0</version>
</dependency>
